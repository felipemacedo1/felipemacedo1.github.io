
# 💻 Terminal Portfolio -## 🎮 Como Usar

### Desktop (Terminal)
1. Abra `index.html` em qualquer navegador
2. Digite `help` para ver todos os 25+ comandos
3. Use setas ↑↓ para navegar no histórico
4. Tab para auto-complete inteligente
5. `Ctrl+L` para limpar, `Ctrl+C` para cancelar

### Mobile (BIOS)
1. Acesse automaticamente ou vá direto para `mobile.html`
2. Navegue pelos menus touch-friendly organizados
3. Acesse "Desktop Version" para forçar terminal
4. Boot sequence animada na primeira visita

**✨ Detecção automática** - dispositivos móveis são redirecionados para a interface BIOS

> 🚀 Portfólio interativo dual: Terminal retrô para desktop + Interface BIOS Android para mobile

Um portfólio web único que oferece duas experiências distintas:
- **Desktop**: Terminal Linux/Unix retro com 25+ comandos interativos e 6 temas
- **Mobile**: Interface BIOS Android moderna e touch-friendly com Material Design

Desenvolvido com HTML, CSS e JavaScript vanilla - **sem dependências externas!**

## 🎯 Acesso Rápido

- **🖥️ Versão Desktop**: [felipemacedo1.github.io/static-port/](https://felipemacedo1.github.io/static-port/)
- **📱 Versão Mobile**: [felipemacedo1.github.io/static-port/mobile.html](https://felipemacedo1.github.io/static-port/mobile.html)
- **🔄 Auto-redirect**: Detecção automática de dispositivo

## 🏗️ Arquitetura Técnica

### 📁 Estrutura Modular
```
static-port/
├── index.html              # Entrada desktop
├── mobile.html             # Entrada mobile
├── css/                    # Estilos modulares
│   ├── main.css           # Desktop principal
│   ├── mobile-bios.css    # Mobile BIOS
│   └── [modules].css      # Módulos organizados
├── js/                     # JavaScript modular
│   ├── main.js            # Entry point com detecção
│   ├── mobile-bios.js     # Interface BIOS
│   ├── core/              # Funcionalidades centrais
│   ├── commands/          # Comandos organizados
│   ├── features/          # Features específicas
│   └── utils/             # Utilitários
└── docs/                   # Documentação
```

## 🔄 Integração Dual

### Detecção Automática
- **User Agent** - Identifica dispositivos móveis
- **Touch Detection** - Verifica capacidade de toque
- **Screen Size** - Analisa dimensões da tela
- **Override** - `?desktop=true` força desktop no mobile

### Navegação Entre Versões
- **Mobile → Desktop**: Botão "Desktop Version"
- **Desktop → Mobile**: Redirecionamento automático
- **URLs diretas** para cada versão
- **Configuração persistente** via localStorage
